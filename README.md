# 云膳供应链测试系统

面向影刀 RPA 演示的餐饮供应链测试网站，界面参考企业 ERP/金蝶云星空风格。

## 主要功能

- 经营工作台：门店、库存、客流预测与待办概览。
- 门店库存：读取 6 家门店的模拟金蝶数据，支持门店、关键词、动销等级和库存风险筛选及 CSV 导出。
- 智能订货：选择门店、配送周期和订货数量，自动计算预计成本并提交中央厨房订单。
- 采购订单：查询已提交订单及模拟历史订单。
- 门店档案、供应商管理、系统设置。

关键输入框、按钮、结果字段均使用稳定 HTML `id`，便于影刀元素捕获。订单保存在浏览器 `localStorage` 中，仅用于演示。

## 本地运行

双击 `start_server.bat`，访问 `http://127.0.0.1:8899/`。
