# 企业自动化演示站

这是一个适合影刀 RPA 捕获与流程演示的纯静态电商后台，可直接部署到 Netlify。

## 本地运行

双击 `start_server.bat`，浏览器会打开 `http://127.0.0.1:8899/`。

## Netlify 部署

- Build command：留空
- Publish directory：`.`
- 项目已包含 `netlify.toml`

## 常用演示流程

1. 选择“天猫旗舰店”和“衣物清洁”。
2. 点击“查询数据”。
3. 读取结果数量和核心指标。
4. 点击“导出 CSV”，验证下载文件。

## 完整功能模块

- 首页看板：经营指标、实时趋势、待办任务和渠道表现。
- 商品销售报表：多条件筛选、结果统计和 CSV 导出。
- 活动复盘：目标进度、流量归因、活动明细和新建复盘。
- 库存监控：库存预警、仓库容量、补货任务和操作日志。
- 渠道管理：店铺目标、连接状态和渠道数据同步。
- 系统设置：通知开关、同步频率、日报邮箱和演示数据恢复。

所有模块均通过页面内路由切换，并支持带哈希的直达地址，例如 `/#inventory`。

筛选框和按钮均保留稳定的 HTML `id`，方便影刀重复捕获。后续为其他企业改版时，优先替换品牌文案、颜色和示例数据，不修改已有控件 `id`。
