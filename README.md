# 连锁餐饮供应链管理系统（演示版）

面向影刀 RPA 与 AI 智能体测试的公开演示网站，采用企业 ERP / 金蝶云星空风格。项目为纯静态前端，不包含登录页，不使用 Supabase 或其他后端数据库。

## 主要功能

- 经营看板：门店、库存、客流预测与待办概览。
- 门店经营数据：每家门店最近 60 天确定性模拟数据，支持日期筛选、分页和 CSV 导出。
- 库存与在途：6 家门店、90 项模拟金蝶库存，支持门店、物料、动销等级、库存状态筛选和导出。
- 菜品与 BOM：菜品资料、售价、销量占比、标准用量与损耗率。
- 新建配送订单：动态增加/删除原材料，校验 request_id 幂等性，自动计算成本并生成中央厨房订单。
- 中央厨房订单：按订单号、request_id、门店、中央厨房、状态和配送日期查询，支持详情、取消及导出。
- 门店档案、操作日志和演示数据管理。

关键输入框、按钮、表格和结果字段均使用稳定 HTML `id` / `data-testid`，便于影刀元素捕获。新增订单和操作日志保存在浏览器 `localStorage` 中，仅用于演示。

默认订单示例：广州天河店、华南中央厨房、配送周期 2026-07-24 至 2026-07-30、配送地址“测试地址-天河商圈”、预计成本 ¥8,326.50、提交状态“已提交”。订单号规则为 `CK-区域-YYYYMMDD-6位流水号`。

## 本地运行

双击 `start_server.bat`，访问 `http://127.0.0.1:8899/`。
